# Changelog

所有版本更新记录遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 规范。

## [1.0.0] - 2024-03-27

### 新增
- DAO模式光盘刻录（Disc At Once）
- SHA256自动校验功能
- 完整审计日志系统
- USB刻录机自动检测
- 前台服务保活机制
- Material Design 3 界面
- 审计日志ZIP导出
- 多设备芯片支持

### 技术特性
- Kotlin Coroutines + Flow 响应式编程
- Jetpack Compose UI框架
- USB Host API + SCSI命令集
- Android 10+ (API 29+) 支持

### 安全
- 数据完整性校验
- 结构化审计日志
- 错误码标准化
